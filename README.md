# SkyJumpers
SkyJumpers is a 2D platformer developed in Unity where players control a fallen Skyjumper navigating floating islands and ancient celestial structures. The game focuses on precise movement, parkour mechanics, and challenging jumps as players ascend through a fragmented fantasy world in the sky

## Features
- Character movement and jumping
- Collectible relics
- Score system
- Checkpoints
- Level completion system

## Controls
- A / D → Move
- Space → Jump
- Shift + Space → Long jump
